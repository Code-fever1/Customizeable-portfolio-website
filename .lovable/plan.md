

# Premium Developer Portfolio Website

## Design Vision
A futuristic, glassmorphism-powered portfolio with dark mode by default, featuring animated gradients, particle effects, parallax scrolling, and premium micro-interactions throughout.

## Color & Style
- **Dark theme default** with blue/purple gradient accents and optional light mode toggle
- Glassmorphism cards with backdrop blur and soft borders
- Glow effects, soft shadows, and gradient mesh backgrounds
- Premium typography with large bold headings

## Sections

### 1. Sticky Navbar
- Transparent with blur backdrop on scroll
- Smooth scroll navigation links
- Dark/light mode toggle
- Mobile hamburger menu

### 2. Hero Section (Full Screen)
- Animated gradient mesh background with floating particle effects
- Profile photo with glowing ring frame and hover animation
- Large bold name with animated typing effect cycling through roles (Web Developer | Problem Solver | Tech Enthusiast)
- Three CTA buttons: View Projects, Download CV, Contact Me
- Custom cursor glow effect that follows the mouse
- Parallax depth on background elements

### 3. About Section
- Split layout with bio card (glassmorphism)
- Personal traits with animated icons
- Stats counters (projects completed, years experience, etc.) with count-up animation on scroll
- Scroll-reveal entrance animations

### 4. Skills Section
- Categorized skill cards (Frontend, Backend, Tools)
- Animated progress bars that fill on scroll
- Technology icons with hover glow effects
- Staggered entrance animations

### 5. Projects Section
- Responsive grid of project cards
- Each card: screenshot preview, title, description, tech stack badges, Live Demo + GitHub buttons
- Hover overlay animation with slide-up details
- Category filter tabs (All, Web, Mobile, etc.)
- Parallax tilt effect on hover

### 6. Services / Experience Section
- Icon-based service cards (Websites, CMS, SaaS, APIs)
- Clean grid layout with glassmorphism styling
- Hover animations with glow accent

### 7. Contact Section
- Modern contact form with animated floating labels
- Email and social media links with icon hover effects
- Form submission success animation (visual only, no backend)
- Glassmorphism form card

### 8. Footer
- Minimal design with social icons
- Copyright text
- Back-to-top button with smooth scroll

## Advanced Visual Effects
- **Animated gradient background**: Continuously shifting gradient mesh across sections
- **Particle effects**: Subtle floating particles in hero and background
- **Parallax scrolling**: Multi-layer depth effect on scroll
- **Custom cursor glow**: Radial gradient glow following the cursor
- **Section transitions**: Scroll-triggered reveal animations (fade, slide, scale) on every section
- **Smooth scrolling**: Native smooth scroll behavior throughout

## Responsive Design
- Fully responsive across mobile, tablet, and desktop
- Adapted layouts and font sizes per breakpoint
- Touch-friendly interactions on mobile

## Content
- Placeholder name, bio, projects, and skills (easily customizable later)
